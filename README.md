
# NHL Pool (Any Server) — v5.3

- Pelaajahaku: Records API
- Pisteet: api.nhle.com (skater/goalie summary), nimi api-web.nhle.com
- UI: kausivalitsin-napit, Ranking-näkymä, mobiilissa pelaajataulukko muuttuu korteiksi

## Ajo
```bash
npm install
npm start
# http://localhost:3000
```
